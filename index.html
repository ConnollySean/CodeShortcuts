Helpful Keyboard Shortcuts

ctrl+enter = create next line blank and move to it w/o bringing rest of line with

alt+up/down = move whole row up or down

ctrl+x = cut (if nothing highlighted it cuts whole line!)

ctrl+c = copy (if nothing highlighted it copys whole line!)

ctrl+v = paste

ctrl+shift+k = delete(entire line)

ctrl+f = find

ctrl+h = replace

ctrl+shift+l = select all occurences of current selection
